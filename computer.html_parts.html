Computer

Computer is an electronic device for storing and processing data, typically in binary form, according to instructions given to it in a variable program

The basic parts of a computer are:

  • Monitor: The monitor is the screen that displays the output, allowing users to see and interact with the computer's graphical user interface.
  • CPU (Central Processing Unit): The CPU is the brain of the computer, responsible for executing instructions and processing data.
  • Keyboard: The keyboard is an input device used for typing and entering commands or text into the computer.
  • Mouse: The mouse is another input device used to navigate the computer's graphical interface, making it easier to interact with the computer.
