Basic HTML

Introducing HTML
  • HTML (Hypertext Markup Language) is the set of markup symbols of codes inserted in a file intended for display on a web browser. The markup tells the web browser how to display a web page's content for the user. It is a markup language that structures the content on the web, defining elements and their attributes to create a structured document.
  • The primary language used on the Web, which contains a set of elements commonly referred to as tags that defines a document and guides the browser in displaying the document. HTML allows developers to structure content and create well-formatted documents that include text, images, links, forms, and other elements.
